# SunKV 当前版本绝对性能报告（2026-04-24，修订版）

## 1. 测试目的

在当前主线版本上产出一组可复现的**绝对性能基线**，补齐：

- 机器与环境信息
- 重复实验与波动统计
- 标准化混合场景口径

## 2. 测试前处理

- 清空持久化文件：
  - `data/wal/wal2.bin`
  - `data/wal/wal2.bin.*`
  - `data/snapshot/snapshot2.bin`
- 冷启动恢复确认：
  - `snapshot not found`
  - `wal probe ... bytes=0`
  - `wal replay applied: mutations=0`

## 3. 机器与软件环境

- 时间：`2026-04-24T16:07:23+08:00`
- OS：`Linux 6.17.0-22-generic x86_64`（Ubuntu）
- CPU：`AMD Ryzen 9 7945HX with Radeon Graphics`（虚拟化环境，`lscpu` 显示 4 vCPU）
- 内存：`3.3 GiB`（可用约 `1.4 GiB`）
- 编译器：`g++ 15.2.0`
- CMake：`3.31.6`
- 压测工具：`redis-benchmark 8.0.2`
- 服务启动命令：
  - `./build/server/sunkv --config ./sunkv.conf --port 6380`

## 4. 测试方法（标准化）

- 主机：`127.0.0.1:6380`
- 公共参数：`-n 200000 -c 100 -P 16`
- 每个场景重复 **3** 次，报告中位数（median）与标准差（stddev）

场景定义：

1. `SET`：`redis-benchmark -t set`
2. `GET`：`redis-benchmark -t get`
3. 混合（标准化 1:1）：`redis-benchmark -t set,get -q`  
   - 口径说明：每轮同时执行 set/get，分别记录两者吞吐；混合总吞吐 = `SET_rps + GET_rps`
